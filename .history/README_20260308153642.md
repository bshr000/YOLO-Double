# YOLO-Double

基于 PyTorch 的双模态目标检测项目（Visible + Infrared），旨在完整打通多模态目标检测框架的基本流程。
整体思路参考YOLO-Single。

## 1. 项目简介

本项目面向 YOLO 格式标注（`class cx cy w h`，归一化坐标），核心特点：

- 双分支骨干网络（Visible 分支 + Infrared 分支），两路特征采用轻量的逐元素相加（Add）进行融合。
- 多尺度特征融合（PAN-FPN）
- DFL 回归分支与 Anchor-Free 风格预测
- 训练支持 EMA、AMP、梯度裁剪
- 提供训练、验证、推理三条主流程脚本

## 2. 仓库结构

- `train.py`：模型训练入口
- `val.py`：验证评估（mAP@0.5 与 mAP@0.5:0.95）
- `inference.py`：离线推理与可视化保存
- `dataset.py`：双模态数据读取与增强
- `model_dual.py`：双分支检测模型定义
- `loss.py`：检测损失定义（CIoU + DFL + SimOTA）
- `utils.py`：工具函数（EMA、NMS、checkpoint）
- `config.yaml`：训练与数据配置

## 3. 环境安装

建议 Python 3.10。

```bash
pip install -r requirements.txt
```

## 4. 数据准备

默认代码通过路径替换规则进行双模态配对：`visible -> infrared`。  
也就是说，可见光图片路径中需要包含 `visible` 目录名，并且红外图片与其同名。

示例目录结构：

```text
your_dataset/
  images/
    visible/
      000001.png
      000002.png
    infrared/
      000001.png
      000002.png
  labels/
    visible/
      000001.txt
      000002.txt
  train.txt
  val.txt
```

其中：

- `train.txt` / `val.txt` 内容为 **visible 图像路径列表**
- 标签使用 YOLO 格式：`class cx cy w h`

## 5. 配置说明

在 `config.yaml` 中修改以下关键字段：

- `data.train_images`
- `data.train_labels`
- `data.val_images`
- `data.val_labels`
- `data.num_classes`

## 6. 使用方式

### 6.1 训练

```bash
python train.py --config config.yaml
```

### 6.2 验证（mAP）

```bash
python val.py --verbose --config config.yaml --checkpoint checkpoints/best_model.pth
```

### 6.3 推理

```bash
python inference.py --save-txt --source path/to/images_or_txt --config config.yaml --checkpoint checkpoints/best_model.pth --output inference_output
```
